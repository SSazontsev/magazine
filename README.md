# Проект интернет магазин

1. **Разработать `base.html` шаблон**  
   - Создать базовый шаблон для всех страниц.

2. **Создать форму для регистрации/входа**  
   - Разработать формы для авторизации и регистрации пользователей.

3. **Функция авторизации**  
   - Реализовать логику входа в систему.

4. **Функция регистрации**  
   - Реализовать логику регистрации новых пользователей.

5. **Разработать модели товаров для БД и админки**  
   - Создать модели для товаров, включая категории, размеры и фотографии.

6. **Разработать модели заказов для БД и админки**  
   - Создать модели для заказов, корзины и пользовательских данных.

7. **Добавить возможность прикреплять к продуктам фотографии**  
   - Реализовать загрузку и отображение фотографий товаров.

8. **Оформить приложение каталога**  
   - Создать страницу каталога с отображением всех товаров и информации о них.

9. **Функция фильтрации по категориям**  
   - Добавить фильтрацию товаров по категориям.

10. **Функция фильтрации по размерам**  
    - Добавить фильтрацию товаров по размерам.

11. **Возможность открывать карточки товара**  
    - Реализовать переход на страницу товара из каталога.

12. **Оформить страницу товара (детальную информацию)**  
    - Создать страницу с подробной информацией о товаре.

13. **Функция выбора размера в карточке товара**  
    - Добавить возможность выбора размера на странице товара.

14. **Оформить страницу корзины**  
    - Создать страницу для отображения товаров в корзине.

15. **Функция добавления в корзину товаров**  
    - Реализовать добавление товаров в корзину из карточки товара.

16. **Функция удаления из корзины товаров**  
    - Реализовать удаление товаров из корзины.

17. **Настроить логику работы корзины товаров**  
    - Реализовать заполнение на основе куки, подсчет общей стоимости, количества товаров и очистку корзины.

18. **Форма для заполнения информации о заказе**  
    - Создать форму для ввода данных пользователя при оформлении заказа.

19. **Функция оформления заказов**  
    - Реализовать логику оформления заказов.

20. **Настроить платежную систему Stripe**  
    - Интегрировать Stripe для обработки платежей.

21. **Разработать личную страницу пользователя**  
    - Создать страницу для отображения информации о пользователе и его заказах.
